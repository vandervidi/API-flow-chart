Privacy Policy

API Flow Chart for Confluence

Last updated: March 9, 2026

Overview

API Flow Chart is a Confluence macro that helps teams create and visualize API sequence diagrams. We are committed to protecting your privacy and being transparent about our data practices.

Key Point: API Flow Chart does not collect, store, or process any personal user data. We only store the diagram content you create.

What Data We Store

API Flow Chart stores only the diagram data you explicitly create:

  • Participant names (e.g., "Frontend", "Backend", "API Gateway")
  • API endpoint information you enter
  • Step descriptions you provide

What We Don't Collect

  • No personal information (names, emails, IP addresses)
  • No usage analytics or tracking
  • No cookies or browser fingerprinting
  • No data sharing with third parties

Data Storage & Security

All diagram data is stored securely using Atlassian Forge Storage, which means:

  • Data remains within Atlassian's secure infrastructure
  • Full compliance with Atlassian's data residency requirements
  • Enterprise-grade security and encryption
  • Data is isolated per Confluence instance

Data Retention & Deletion

Your diagram data is automatically deleted when:

  • You remove the API Flow Chart macro from a page
  • The Confluence page containing the macro is deleted
  • The app is uninstalled from your Confluence instance

Your Rights

Since we don't collect personal data, standard data subject rights (access, correction, deletion) apply only to the diagram content you create, which you can edit or delete at any time directly within Confluence.

Changes to This Policy

We may update this privacy policy from time to time. Any changes will be reflected on this page with an updated revision date.
